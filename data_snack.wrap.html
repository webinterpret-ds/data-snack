get_dataframe(df: DataFrame) → DataFrame¶

Uses a data frame to retrieve data from the db. Input data frame should contain all the columns corresponding to keys of selected Entity type.

Parameters:

df – a data frame containing columns with entity keys

Returns:

a full data frame containing all retrieved entities

set_dataframe(df: DataFrame) → List[str]¶

Uses a data frame as a source for creating multiple Entity objects. Data frame needs to follow the schema defined by the Entity. Each row in the data frame defines a single Entity that will be saved in the db.

Parameters:

df – a data frame with entities in a tabular form

Returns:

a list of keys created for entities
