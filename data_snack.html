delete(cls: Type[Entity], key_values: List[Any]) → bool¶

Deletes one entity of Entity type from db based on provided key values. Notice, key is represented as a list of strings, since one Entity can have multiple key fields.

Parameters:

  • cls – Entity type

  • key_values – a list of key values representing the entity

Returns:

True if data were deleted

delete_many(cls: Type[Entity], keys_values: List[List[Any]]) → bool¶

Deletes multiple Entity objects in db.

Parameters:

  • cls – Entity type

  • keys_values – list of keys, each list defines a set key values for one Entity object

Returns:

True if data were deleted

get(cls: Type[Entity] | Type[CompoundEntity], key_values: List[Any]) → Entity | CompoundEntity | None¶

Gets one entity depending on type from db based on provided key values. Notice, key is represented as a list of strings, since one entity can have multiple key fields.

Parameters:

  • cls – Entity or CompoundEntity type

  • key_values – a list of key values representing the entity

Returns:

a retrieved Entity or CompoundEntity object

get_many(cls: Type[Entity] | Type[CompoundEntity], keys_values: List[List[Any]]) → List[Entity | CompoundEntity | None]¶

Gets list of entity objects depending on type from db based on provided key values. Notice, key is represented as a list of strings, since one entity can have multiple key fields.

Parameters:

  • cls – Entity or CompoundEntity type

  • keys_values – list of keys, each list defines a set key values for one entity object

Returns:

a list of retrieved Entity or CompoundEntity objects
