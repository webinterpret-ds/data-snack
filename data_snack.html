delete(cls: Type[Entity], key_values: List[Any]) → bool¶

Deletes one entity of Entity type from db based on provided key values. Notice, key is represented as a list of strings, since one Entity can have multiple key fields.

Parameters:

  • cls – Entity type

  • key_values – a list of key values representing the entity

Returns:

True if data were deleted
